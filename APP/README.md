# EasyAIoT APP模块

EasyAIoT APP 是基于 uni-app 3 构建的跨端移动端管理后台，一套代码编译 H5、微信小程序与原生 App，与 PC 端（WEB）共用同一套后端 API（`/admin-api`），面向移动场景的运维与管理人员提供随时随地管控能力。

## 功能特性

- **跨端部署**: 基于 uni-app 3，一套代码编译 H5、微信小程序与原生 App，无需依赖 HBuilderX，支持命令行开发
- **API 复用**: 与 WEB 模块共用 `/admin-api` 后端接口，OAuth2 双 Token 认证，登录态自动续期，支持多租户切换
- **设备管理**: 支持直连摄像头、GB28181、NVR 多协议统一列表与通道浏览，设备详情内一键启动实时画面预览
- **推流转发**: 支持推流转发任务创建、启停、集群节点状态与多路流地址查看
- **算法任务**: 支持实时/抓拍算法任务列表、启停控制与检测/帧数统计
- **告警中心**: 支持告警事件检索、抓拍图预览与告警录像点播回放
- **模型管理**: 支持模型列表查看与部署状态监控
- **模型推理**: 移动端图片推理工作台，选模型、传图片、看结果
- **模型训练**: 支持训练任务进度监控与一键停止
- **视频监控**: H5 端集成 Jessibuca 低延时播放器，支持 FLV/HLS 实时流与告警录像点播
- **个人中心**: 支持个人资料、账号安全、常见问题、意见反馈、隐私设置与应用配置
- **消息通知**: 支持站内消息检索与详情查看

## 技术栈

- **框架**: uni-app 3 / Vue 3
- **语言**: TypeScript
- **构建工具**: Vite 5
- **UI 框架**: Wot Design Uni
- **样式方案**: UnoCSS
- **状态管理**: Pinia（含持久化）
- **路由**: 约定式路由（uni-pages）
- **HTTP 客户端**: 自研请求封装（支持 API 加解密）
- **视频播放**: Jessibuca / flv.js
- **图表**: ECharts
- **代码规范**: ESLint
